I download the source code of xboot form atmel.com. Although this can be compiled,however, it is always in application area(start from 0x00). So I can not download my main program in the chip, otherwise the bootloader will be erased.

I tried to add "BOOT_SECTION_START= 0X4000

Where have you added the linker option?
Do you use the latest Atmel Studio (6.1 upd 2)?

Try to add
-Wl,--section-start=.text=0x4000
in Linker->Miscellaneous->other linker flags
